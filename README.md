Ghidra build for Kam1n0 &amp; Jarv1s

### :rocket: Installation:
Required: Python 3+. [Optional: OpenJDK 13+]
```bash
pip install ghidrapy
```

```bash
python -m ghidrapy binary_file_to_decompile
```
